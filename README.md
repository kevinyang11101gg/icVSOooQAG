# 项目简介
基于 python+flask 深度学习的中文情感分析系统
含运行教程
【附赠】初稿 + ppt 
【技术栈】
python3.6、flask、mysql
深度学习：TensorFlow 模型
机器学习：Word2Vec 词向量 + SVM 分类器
调用 SVM 模型输出情感标签（“积极” 或 “消极”）

【所需环境】
一定一定使用Python3.6！！
Flask、mysql8

【功能模块】
用户管理、
个人信息查看、
文本分类、
文本管理
等等功能

## 核心功能
- 用户管理
- 个人信息查看
- 文本分类
- 文本管理

## 适用场景
- 对中文文本情感倾向进行分析
- 了解用户反馈的情感倾向
- 优化内容策略

## 技术特点
- 使用 Python 3.6 和 Flask 框架
- 基于 TensorFlow 深度学习模型
- 结合 Word2Vec 词向量和 SVM 分类器
- 支持MySQL数据库存储
